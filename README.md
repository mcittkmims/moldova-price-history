# Moldova Price History

A Vite and React price-history lab app for browsing Moldova store products, comparing current prices, and tracking saved products locally in the browser.

## Scripts

- `npm run dev` starts the local Vite server.
- `npm run build` runs TypeScript checks and creates the production build.
- `npm run preview` serves the production build locally.

## Verification

The transferred app is verified with `npm run build` after the product pages are in place.
